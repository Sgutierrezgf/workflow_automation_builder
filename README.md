# Workflow Automation Builder

## Instrucciones de Uso

### 1. Clonar el repositorio

```bash
git clone https://github.com/Sgutierrezgf/workflow_automation_builder.git
```

### 2. Instalar proyecto
```bash
npm install
```
### 3. Correr proyecto
```bash
npm run dev
```
